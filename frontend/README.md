# Sprint 2 Task 6: Update Property Page (PM)

This sprint adds the **Update Property Page** for Property Managers.

## Features
- Backend integration with `PUT http://127.0.0.1:8000/properties/{id}`
- Editable form fields for:
  - Property name
  - Address
  - Type
  - Rooms
  - Tariff plan
- Form validation and success/error alerts
- Matching UI style with PM Dashboard (Sprint 1)
